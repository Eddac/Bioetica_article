# Bioethics and Artificial Intelligence: Digital Content Analysis on Labiaplasty
This repository contains the supplementary materials for the study **"Bioética e Inteligência Artificial: Análise de conteúdo digital sobre labioplastia"**. The research evaluates the bioethical implications of digital marketing in labiaplasty using Large Language Models (LLMs).

Appendix2.R: The complete computational pipeline developed in R (version 4.2.2). It includes functions for web scraping textual content and visual metadata (alt-text), as well as the integration with the OpenAI GPT-4 API .
